About REVM

REVM is a pure Rust implementation of the Ethereum Virtual Machine (EVM) used by major blockchain projects:

  • Foundry - Smart contract development toolchain
  • Reth - Ethereum execution client
  • Helios - Ethereum light client
  • Alloy - Ethereum development framework

Key Features

  • Blazing Fast - Optimized Rust performance
  • Memory Safe - No undefined behavior
  • Modular Design - Pluggable components
  • No-std Support - Embedded environments
  • Full EVM Compatibility - All opcodes supported

Quick Start

Installation

Add REVM to your Cargo.toml:

[dependencies]
revm = "34.0.0"

Basic Usage

use revm::{
    db::{CacheDB, EmptyDB},
    primitives::{address, U256},
    Evm,
};

fn main() {
    let db = CacheDB::new(EmptyDB::default());
    let evm = Evm::builder()
        .with_db(db)
        .build();
    
    let result = evm.transact().unwrap();
    println!("Gas used: {:?}", result.result.gas_used());
}

Run Examples

git clone https://github.com/bluealloy/revm
cd revm
cargo run --example my_evm
